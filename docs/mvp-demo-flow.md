# Nami MVP Demo Flow

## Purpose

This document describes the current presentable MVP demo flow for Nami Chat.

The goal of this demo is to show that Nami is more than a chat app.

Nami is a Sui-powered gaming identity, reputation, access, moderation, customization, and social protocol.
