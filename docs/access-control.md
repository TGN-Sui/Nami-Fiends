# Nami Access Control v1.0

## Overview

Access Control determines which features, permissions, and responsibilities are available to users within the Nami ecosystem.

Access is determined through:

- Verification Status
- Membership Tier
- Trust Status
- Reputation Status
- Community Roles

---

# Identity States

## Unverified

Entry-level state.

Can:
- Create profile
- Join public channels
- Earn XP
- Earn reputation
- Earn badges
- Participate in public discussions

Cannot:
- Create Guilds
- Issue badges
- Use boosts
- Sponsor Squad Members
- Access premium gated communities

---

# Membership Tiers

## Adventurer

Verification tier.

Obtained through:
- Nami verification subscription
- Approved verification providers (X, SuiNS, future integrations)

Can:
- Access core Nami features
- Subscribe to channels
- Participate in gated communities
- Earn claimable rewards

Cannot:
- Use boosts
- Sponsor Squad Members

---

## Pro

Full-access supporter tier.

Can:
- Use weekly boosts
- Participate in channel discovery influence
- Access advanced community features

---

## Elite

Premium supporter tier.

Can:
- Use expanded weekly boosts
- Sponsor Squad Members
- Access premium cosmetic systems
- Unlock advanced customization

---

# Trust Tiers

Trust is separate from Membership.

Trust is earned.

---

## Trusted Member

Requirements:
- Positive participation history
- Reputation threshold met

Permissions:
- Create Guilds
- Sponsor Squad Members
- Enhanced community influence

---

## Community Champion

Highest trust tier.

Permissions:
- Increased Squad slots
- Expanded discovery influence
- Community leadership recognition

---

# Permission Matrix

| Feature | Unverified | Adventurer | Pro | Elite | Trusted | Champion |
|----------|------------|------------|-----|--------|----------|-----------|
| Public Chat | Yes | Yes | Yes | Yes | Yes | Yes |
| Gated Chat | No | Yes | Yes | Yes | Yes | Yes |
| Guild Creation | No | No | No | No | Yes | Yes |
| Boost Channels | No | No | Yes | Yes | Yes | Yes |
| Sponsor Squad Members | No | No | No | Yes | Yes | Yes |
| Issue Official Badges | No | No | No | No | Limited | Approved Only |

---

## Core Principle

Influence is earned.

Access can be purchased.

Trust cannot.
