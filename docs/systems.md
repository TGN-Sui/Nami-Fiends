# Nami Systems Overview v1.0

## Identity System

Core wallet-bound identity.

Stored on Sui.

---

## Passport System

Progression system containing:

* Level
* XP
* Badges
* Prestige
* Puzzle pieces

---

## Trust System

Determines social influence:

* Verified Member
* Adventurer
* Pro
* Elite
* Trusted Member
* Community Champion

---

## Reputation System

Earned status system:

* Newbie
* Gamester
* Goblin
* Goonie
* Fiend

---

## Boost System

Weekly community influence signal:

* Limited per user per cycle
* Caps per channel
* Resets weekly

---

## Squad System

Trust-based endorsements from higher-tier users to others.

---

## Guild System

Persistent community structures with reputation and governance.

---

## Discovery System

Ranking engine that determines visibility of:

* channels
* guilds
* hubs

Based on:

* engagement
* trust
* boosts
* activity
